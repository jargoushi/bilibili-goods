# B站好物工业化生产操作台（人机协同版）

---

# 一、整体架构（人机协同工厂）

```
            ┌────────────────────────┐
            │   数据输入（对标视频链接） │
            └────────┬───────────────┘
                     ↓
    ┌────────────────────────────────────┐
    │   ① 数据采集层（全自动）             │
    │   视频解析 / 链接提取 / 佣金查询      │
    │   商品参数 / 统一数据库               │
    └────────────┬───────────────────────┘
                 ↓
    ┌────────────────────────────────────┐
    │   ② 数据结构化层（半自动）           │
    │   脚本库 / 卖点标签 / 产品评分        │
    └────────────┬───────────────────────┘
                 ↓
    ┌────────────────────────────────────┐
    │   ③ 决策辅助层（人机协同）           │
    │   选品推荐 / 对标差异分析             │
    └────────────┬───────────────────────┘
                 ↓
    ┌────────────────────────────────────┐
    │   ④ 内容生成层（半自动）             │
    │   脚本 / TTS配音 / 剪辑辅助 / 封面    │
    │   蓝链预设                           │
    └────────────┬───────────────────────┘
                 ↓
    ┌────────────────────────────────────┐
    │   ⑤ 人工质检 + 发布 + 运营          │
    │   审核 / 微调 / 发布 / 评论运营       │
    └────────────────────────────────────┘
```

核心理念一句话：

👉 **机器负责"搬砖 + 结构化"，人负责"判断 + 差异化"。**

**技术栈**：Python全栈（FastAPI + Streamlit）
**AI服务**：国内免费AI为主（DeepSeek、豆包、腾讯元宝）
**TTS方案**：高仿真AI配音（ChatTTS / GPT-SoVITS / CosyVoice）
**数据存储**：SQLite（支持导出Excel）

# 二、模块拆解（直接对应你可以做的功能）

## ① 数据采集层（优先做，收益最大）

👉 目标：把"找对标 + 抄作业"变成自动流程

---

### 对标视频解析器

输入：B站视频链接
输出：

* 标题
* 封面图
* 视频文件
* 字幕文本（自动提取）

实现：bilibili-api / yt-dlp 下载 + B站字幕API / Whisper本地识别

---

### 评论区商品提取器（重点）

功能：

* 抓取评论区（重点置顶评论）
* 提取商品链接
* 自动分类：京东 / 淘宝 / 天猫
* 短链还原为完整商品链接

实现：B站评论API + 正则匹配 + 短链还原

👉 这个在文章里已经有人用影刀跑通

---

### 佣金查询系统

功能：

* 批量查询京东联盟 / 淘宝联盟佣金
* 返回：
  * 商品ID
  * 佣金比例
  * 预估收益

实现：京东联盟API / 淘宝联盟API（需申请账号）

---

### 商品信息抓取 + OCR

功能：

* 爬取京东/淘宝详情页图片
* OCR识别产品参数
* AI提炼"观众听得懂的"卖点和使用场景
* 输出结构化数据

输出示例：

```json
{
  "产品名": "",
  "价格": "",
  "核心卖点": [],
  "使用场景": [],
  "参数": {}
}
```

实现：requests / playwright 爬取 + PaddleOCR + AI整理

---

### 统一数据库（核心资产）

存储：SQLite数据库（替代Excel/Notion，支持导出Excel）

字段：

* 商品名
* 类目
* 佣金
* 视频来源
* 卖点
* 参数
* 标签（人群/场景/需求）

功能：筛选 / 排序 / 标签过滤 / 导出Excel

👉 所有模块的数据汇聚点，越用越有价值

---

## ② 数据结构化层（让系统"越来越聪明"）

👉 本质：把"拆解过的东西变资产"

---

### 脚本库

* 将对标视频脚本拆解为可复用组件：
  * 开头钩子
  * 转折
  * 卖点表达
  * 结尾引导

👉 后续视频可直接组合复用，不从零开始

---

### 卖点标签系统

维度：

* 场景：宿舍 / 户外 / 上班
* 人群：学生 / 打工人 / 宝妈
* 需求：省钱 / 提升效率 / 舒适

👉 这就是文章里鹿书野说的：
**周期 × 群体 × 需求**

功能：给商品和脚本打标签，支持按标签组合检索

---

### 产品评分模型（简单版）

给每个商品打分：

* 佣金（权重30%）
* 卖点数量（权重20%）
* B站竞争度（权重30%）
* 评论热度（权重20%）

输出：推荐优先级排序

👉 让选品从"凭感觉"变成"看数据"

---

## ③ 决策辅助层（关键，不要全自动）

👉 不是让AI选，而是"帮你更快选"

---

### 选品推荐器

输出：

* 今日推荐品
* 可做视频方向
* 对标视频参考

结合：热点日历（季节/节日/开学季等周期热点）+ 产品评分

---

### 对标差异分析

输入：对标视频（已拆解的脚本+数据）
输出：

* 对方卖点结构
* 可优化点
* 可差异化方向

实现：AI分析对标脚本，对比产品数据库，找出空白机会

---

## ④ 内容生成层（效率爆炸的地方）

👉 这一层直接决定你产能

---

### 脚本生成器（半自动）

输入：

* 商品数据
* 卖点
* 对标脚本风格

输出：

* 口播稿初稿

实现：AI（DeepSeek）+ 脚本模板 + 产品参数自动填充
编辑：Web端脚本编辑器，支持润色、预览、版本管理

👉 你负责改 20%

---

### 高仿真TTS配音

定位：成片级配音方案，追求接近真人的自然度

技术选型（按推荐优先级）：

1. **ChatTTS** — 开源免费，支持笑声/停顿等细粒度控制，中文效果好
2. **GPT-SoVITS** — 开源，支持声音克隆（用少量真人录音训练专属音色）
3. **CosyVoice（阿里）** — 开源，多语言，自然度高
4. **MiniMax语音克隆** — 商用级效果，有免费额度

功能：

* 支持多种音色切换（男/女/不同风格）
* 支持SSML标记控制语速/停顿/情感
* 声音克隆：用5-10分钟真人录音训练专属音色（GPT-SoVITS）
* 配音预览：生成后可试听，不满意重新生成

输出：WAV/MP3音频文件 + 带时间戳的字幕文件（SRT）

---

### 剪辑辅助（素材整理 + 剪辑清单）

功能：

* 按脚本段落自动整理素材文件夹（PPT图片按顺序命名）
* 生成结构化剪辑指引（每段 → 对应素材 → 时间估算 → 转场建议）
* 自动生成字幕文件（SRT，基于TTS时间戳）
* 商品宣传视频下载（京东详情页视频）作为Broll素材

👉 人工粗剪从1小时 → 30分钟

> 注：剪映项目文件格式为私有未公开格式，直接生成草稿技术风险高。
> 如后期需要可作为进阶功能单独攻关。

---

### 封面生成器

输入：视频标题 + 品类
输出：3-5个封面方案（16:9 + 4:3 两版）

实现：Pillow/PIL 模板合成，预设多套封面模板

---

### 蓝链 + 评论预设

功能：

* 根据选品列表自动生成格式化蓝链评论文本
* 支持蛋哥的"预设蓝链"方法（提前在旧视频下发评论获取链接）
* 生成发布checklist（标题/标签/简介/封面/章节/蓝链）

---

## ⑤ 人工质检 + 发布 + 运营层（必须保留）

你只做3件事决定上限：

1. 选不选这个品
2. 改不改这个表达
3. 发不发这个视频

👉 这三件事决定你上限

---

### 评论运营助手（新增 — 核心转化模块）

> 鹿书野的文章揭示：视频脚本写了4000字，评论区回复了2-3万字。
> 评论区是出单的核心，必须有专门模块支持。

功能：

* 定时监控新评论，按类型分类（求推荐/问参数/比价/售后/闲聊）
* AI根据产品知识库生成回复建议草稿
* 人工审核后发送（核心转化环节，不能全自动）
* 数据看板：播放量/点赞/评论/出单/GMV/佣金趋势
* 运营提醒：3天自然流观察节点、运营数据补量提醒
* 收益报表：按品类/视频/时间段统计

# 六、核心工作流（系统跑通后的使用流程）

```
人工：选定品类方向
  ↓
① 自动：爬取对标视频 → 提取商品链接 → 查询佣金 → 写入数据库
  ↓
② 自动：抓取商品详情 → OCR参数 → AI提炼卖点 → 产品评分
  ↓
③ 人机：系统推荐选品方案 → 人工拍板确认
  ↓
④ 半自动：AI生成脚本初稿 → 人工润色20%（30分钟）
  ↓
④ 自动：高仿真TTS配音 → 生成字幕 → 整理素材 → 生成封面 → 生成蓝链
  ↓
人工：剪辑视频（60-90分钟，有剪辑指引和整理好的素材）
  ↓
⑤ 人工：发布视频 + 粘贴蓝链（按发布清单操作）
  ↓
⑤ 半自动：监控评论 → AI生成回复建议 → 人工审核发送
```

**预估效果**：每个视频从12小时 → 约2.5-3小时人工投入
