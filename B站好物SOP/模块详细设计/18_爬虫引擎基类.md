# 18 爬虫引擎基类

## 1. 模块定位

为各抓取模块提供统一的请求、代理、限流、反爬处理能力。

对应代码目录：
- `app/services/crawler_engine.py`

## 2. 核心能力

- 请求会话管理
- 代理池切换
- 重试与退避
- User-Agent 与指纹策略
- 统一错误码

## 3. 抽象接口

- `fetch_text(url, headers)`
- `fetch_json(url, params)`
- `fetch_binary(url)`
- `run_browser_task(task)`

## 4. 模块依赖

- 视频解析
- 评论抓取
- 商品详情抓取
- Broll 素材下载

## 5. 验收标准

- 各模块不再重复实现重试和反爬逻辑。
- 网络异常可追踪并可重放。
